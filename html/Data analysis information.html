Data analysis information

Written by Lela Andrews

Use of this Linux system for data analysis. QIIME 1.9.1 and QIIME 2 are installed within their own python environments. You can only use QIIME commands or QIIME-dependent scripts (e.g. akutils) within the respective environment. Call either environment from the command line with the alias "q1" or "q2". Deactivate either environment with the alias "deac".

Commands for managing QIIME environments:

  • Activate QIIME 1.9.1: q1
  • Activate QIIME 2: q2
  • Return to base environment: deac
